Entendendo o HTML

(voltar)

O HTML é uma linguagem de marcação, diferente de uma linguagem de programação, ele é utilizado para fazer sites, blogs, todo essa parte de desenvolvimento web. Nesse sentido, a sua criação foi pelo britânico Tim Berners-Lee, em 1989. O hyper texto, é formado pelo html, head e body, essas são principais no começo. Uma agregada na outra.
